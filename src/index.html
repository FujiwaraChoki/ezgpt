How it works

ezgpt is an easy-to-use app that allows you to use GPT-3 in your everyday life. it's as simple as following these steps:

  • Copy a question or a text
  • Press Control+C twice
  • Enjoy the answers which are now in your clipboard!

To get started, just type in a question and click the "Ask GPT-3" button. You can also use the "Clear" button to clear the text box.

ezgpt uses the OpenAI API to get answers from GPT-3. You can read more about the OpenAI API here.
